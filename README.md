# AWS EC2 Web Server Deployment

## Overview
Deployed a personal portfolio website on AWS using EC2 and Apache.

## What I Did
- Launched an EC2 instance (Amazon Linux)
- Configured security groups (HTTP, SSH)
- Connected via SSH
- Installed and configured Apache web server
- Deployed a custom HTML website

## Technologies Used
- AWS EC2
- Linux (Amazon Linux)
- Apache (httpd)
- Networking (Security Groups)

## Result
Successfully hosted a live website accessible via public IP.

## What I Learned
- How to provision and manage cloud compute resources
- Basics of Linux server administration
- Web server setup and deployment
- Cloud networking fundamentals
